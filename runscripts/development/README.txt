These scripts work from project root directory only. Copy them there to use them.

assembleApp.bat assembles a single app; it is useful when fixing assembly errors.
assembleKernel.bat assembles the kernel only; it is useful when doing kernel-only work.
assembleLoader.bat assembles the boot loader and reports its byte size.
buildAndRun.bat requires boot_snowdrop.bxrc to be in the project root directory.
