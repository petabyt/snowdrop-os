These scripts work from project root directory only. Copy them there to use them.

debug_snowdrop.bxrc - the configuration file for debug-mode Bochs

debugRun.bat - starts Bochs in debug mode. Requires the above .bxrc file be in the same folder

debugAssembleApp.bat - must be copied to and run from the root directory. It assembles and generates a list file for the specified app
